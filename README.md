DeepClaim AI

DeepClaim AI is a forensic AI platform that verifies the authenticity of insurance claim images.
It analyzes uploaded claim photos using computer vision, image forensics, and similarity search to detect manipulation, synthetic imagery, and recycled incident photos.

The system produces an Authenticity Score along with visual evidence highlighting suspicious regions.

Overview

DeepClaim AI automates claim image verification by combining forensic analysis and modern AI models.
The platform processes claim images through multiple detection layers and provides an explainable fraud assessment.

Key capabilities include:

Image tampering detection

AI-generated image identification

Damage region segmentation

Historical claim similarity search

Visual fraud explanations

The system is designed to support insurance adjusters and fraud investigation teams by accelerating claim review and improving detection accuracy.

Core Capabilities
Image Forensics

Detects visual manipulation using compression artifact analysis and pixel-level inconsistencies.

Techniques include:

Error Level Analysis (ELA)

Noise pattern analysis

Copy-move detection

Region-Level Inspection

The system isolates potential damage areas before performing forensic analysis to improve detection accuracy.

Segmentation models identify and isolate suspicious regions in the image.

AI Image Detection

Detects images generated by modern generative models using frequency-domain analysis and structural inconsistencies.

Historical Claim Search

Claim images are converted into vector embeddings and compared against a historical claims database to identify reused or duplicated images.

Explainable Fraud Assessment

DeepClaim AI produces a structured fraud report including:

Authenticity score

Fraud signals

Visual heatmap highlighting suspicious areas

Similar historical claims

Workflow

Claim Submission

A claim image is uploaded through the dashboard.

Metadata Inspection

The system extracts metadata such as capture device, timestamp, and editing signatures.

Forensic Analysis

Compression artifacts and noise inconsistencies are analyzed to detect manipulation.

Damage Segmentation

Potential damage areas are identified and segmented for targeted inspection.

AI Image Detection

The system evaluates whether the image contains synthetic artifacts.

Similarity Search

The claim image is compared against previously indexed claims.

Fraud Scoring

Signals from each module are aggregated into a final authenticity score.

Installation

Clone the repository:

git clone https://github.com/your-username/deepclaim-ai

Install dependencies:

pip install -r requirements.txt

Run backend:

uvicorn backend.main:app --reload

Run frontend:

npm install
npm run dev
